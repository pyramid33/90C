# Inicio Rapido

1. Instala Python

2. Descarga el .rar del bot o clonalo en tu visualizador de codigo

3. Abre un terminal en la carpeta del proyecto y instala lo que se necesite:
```bash
cd 90cent
pip install -r requirements.txt
```

4. Configura el archivo `.env.example` con tus keys y renombralo a `.env`

5. Ve a `config.example.py` y cambiale el nombre a `config.py`. La configuracion esta por defecto pero la podéis personalizar. En la linea 241 en `LEADERBOARD_USERNAME` podéis cambiar vuestro nombre en el ranking.

6. Inicia el bot `trading_bot.py`. El dashboard se iniciara automaticamente en http://localhost:5052

Lee `GUIA.md` para información mas detallada y completa.
